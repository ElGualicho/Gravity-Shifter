# Gravity Shifter

Jeu de plateformes avec mécanique d'inversion de gravité.

## Fichiers
- `index.html` + `game.js` + `style.css` → Jeu principal
- `editor.html` + `editor.js` → Éditeur de niveaux

## Assets requis (`assets/`)
- `background.png`
- `platform1.png` à `platform8.png`
- `flag.png`
- `pics.png` (cristaux/piques)
- `walk1.png` à `walk4.png` (animation joueur)

## Contrôles
- **← →** : Déplacement
- **Espace** : Inverser la gravité (au sol)
- **Échap** : Menu
